# Treinamento .Net Core

### Sinopse ###
<p> 
  Treinamento com foco na tecnologia .net core 3.1, persistindo no SQL Server.
</p>

### Tópicos ###

* Controllers.
* Model.
* Views.
* Areas.
* Injeção de dependências.

* Configurações.
* Identity.
* Desenvolvendo uma aplicação MVC core completa.
